# 🎓 DropTrack AI: Student Dropout Prediction and Counselling System

## 📌 Overview
DropTrack AI is an intelligent student retention platform designed to identify students who are at risk of dropping out using Artificial Intelligence and Machine Learning. The system analyzes academic, attendance, behavioural, and socio-economic factors to generate early predictions and provide personalized counselling interventions.

The platform enables educational institutions to proactively support vulnerable students through data-driven insights, automated alerts, and structured counselling workflows.

## 🚀 Key Features

- 🔮 AI-powered student dropout prediction
- 📊 Risk classification (High, Medium, Low)
- 👨‍🏫 Personalized counselling recommendations

## 👥 User Roles

### Admin
- Manage institutions and users
- Monitor overall dropout statistics
- View institutional reports

### School Administrator
- Add and manage student records
- Upload student data through Excel
- Monitor at-risk students
- Assign counsellors and mentors

### Counsellor / Mentor
- Access assigned students
- Schedule counselling sessions
- Track student progress
- Record intervention outcomes

### Student
- View academic analytics
- Access counselling support
- Track attendance and performance
- Interact with AI guidance assistant

## 📊 Machine Learning Models

The following models were evaluated:

- Logistic Regression
- Random Forest
- XGBoost
- Artificial Neural Network (ANN)

XGBoost demonstrated the highest prediction accuracy and was selected as the primary model for deployment.
